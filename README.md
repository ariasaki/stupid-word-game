
# Word Game

a simple Game to improve typing skills.



## Installation

Ubuntu and debain Based GNU-Linux Distros:

```bash
  $ git clone https://github.com/ariasaki/word-game
  $ sudo apt install gcc libcurses-dev
  $ cd word-game
  $ gcc main_unix.c -pthread -lncurses
  $ ./a.out
```
